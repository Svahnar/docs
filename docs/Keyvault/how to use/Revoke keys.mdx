If you want to revoke key in keyvault you can do so by following steps:-

**Note**:- User who had created the key only can delete the key

To revoke a key, follow these steps:

1. Navigate to your **Key Vault**.  
2. Locate the key you want to revoke.  
3. In the key’s row, click on the **Delete** icon.  
4. A confirmation dialog will appear — confirm the action.  

⚠️ **Important:**  
Once a key is revoked, it is **permanently deleted**. You will no longer be able to access or recover it from any location.
